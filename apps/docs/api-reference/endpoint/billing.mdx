---
title: 'Get Billing Balance'
openapi: 'GET /api/v1/billing'
---

Retrieve your current credit balance and latest billing transaction.

## What it returns

- `creditsLeft`: Current available credits.
- `lastUpdated`: Last credits table update timestamp, or `null` if never updated.
- `userId`: User ID linked to your API key.
- `lastBillingTransaction`: Most recent `purchase` or `topup` transaction, or
  `null` if none exists yet.

## Example Request

```bash
curl -X GET 'https://sexyvoice.ai/api/v1/billing' \
  -H 'Authorization: Bearer sk_live_xxxxxxxxxxxxxxxxxxxxxxxxxxxxxxxx'
```

## Debugging

All responses include a `request-id` header. Store it in your logs and include
it in support requests when troubleshooting.
